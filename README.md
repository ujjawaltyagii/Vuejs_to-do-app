# to-do-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### json server
```
npm install json-server --save-dev
```

### install AXIOS
```
npm install axios
```

```
npx json-server --watch db.json --port 3000
```
